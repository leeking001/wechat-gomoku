## 📱 提交到微信平台步骤

### 第一步：上传代码

1. 在微信开发者工具中
2. 点击工具栏右上角的"上传"按钮
3. 填写版本号（如 1.0.0）
4. 填写项目备注
5. 点击"上传"

### 第二步：在微信小程序后台设置

1. 访问 [微信小程序后台](https://mp.weixin.qq.com/)
2. 登录你的小程序账号
3. 进入"版本管理" → "开发版本"
4. 点击"提交审核"

### 第三步：填写审核信息

需要填写的信息：

#### 基本信息
- **小程序名称**：五子棋
- **小程序简介**：一个功能完整的微信小程序五子棋游戏，支持与 AI 对战
- **小程序类目**：游戏 → 休闲游戏
- **小程序标签**：五子棋、游戏、AI

#### 功能介绍
```
本小程序是一款五子棋游戏，主要功能包括：

1. 与 AI 对手进行五子棋对战
2. AI 具有智能决策能力，能够进攻和防守
3. 完整的游戏规则实现，支持检测胜负
4. 游戏统计功能，记录胜/负/平局数据
5. 美观的木纹棋盘设计和逼真的棋子样式
6. 游戏规则说明和使用提示

游戏规则：
- 黑棋先手（玩家为黑棋）
- 轮流在棋盘上放置棋子
- 先连成 5 个同色棋子（横、竖、斜）即为获胜
- 棋盘满了还没有人赢则为平局
```

#### 隐私政策
```
本小程序致力于保护用户隐私：

1. 本程序不收集任何个人信息
2. 游戏数据仅保存在本地，不上传到服务器
3. 不使用用户位置信息
4. 不访问用户文件
5. 仅使用必要的微信基础能力
```

#### 操作步骤
```
1. 点击棋盘空位放置黑棋
2. AI 自动放置白棋
3. 轮流下棋直到游戏结束
4. 点击"重新开始"开始新局
5. 点击"游戏规则"查看详细说明
```

### 第四步：等待审核

- 微信通常在 24 小时内完成审核
- 审核通过后会收到邮件通知
- 审核不通过也会说明原因，可以修改后重新提交

### 第五步：发布上线

审核通过后：
1. 登录微信小程序后台
2. 进入"版本管理" → "审核版本"
3. 点击"发布"按钮
4. 小程序正式上线

## 🔍 常见问题解决

### 问题 1：上传时显示"签名校验失败"
**解决**：
- 检查 project.config.json 中的 appid 是否正确
- 在微信开发者工具中重新登录
- 清除开发者工具缓存后重试

### 问题 2：审核被拒原因常见于
- **功能不完整**：确保游戏完全可玩
- **隐私政策不清**：补充清晰的隐私政策
- **体验不好**：优化 UI 和响应速度
- **违规内容**：确保没有不适当内容

### 问题 3：如何更新已上线的小程序
1. 修改 app.json 中的 version 字段
2. 上传新版本
3. 提交审核
4. 审核通过后点击发布

## 📝 审核建议

### 加强点

1. **功能演示**
   - 在操作步骤中详细说明如何玩
   - 提供截图展示游戏界面

2. **合规性**
   - 明确隐私政策
   - 说明不收集用户数据
   - 说明如何使用存储

3. **质量保证**
   - 提供完整的功能列表
   - 说明已测试的设备
   - 列出支持的微信版本

### 优化建议

1. **性能优化**
   - 检查内存使用

   - 考虑添加排行榜
