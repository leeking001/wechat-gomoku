## 🎯 第三阶段：提交审核（30 分钟）

### 步骤 1：登录微信小程序后台

1. 打开浏览器访问：https://mp.weixin.qq.com/
2. 用微信扫码登录
3. 进入小程序管理后台

### 步骤 2：进入版本管理

1. 左侧菜单 → "版本管理"
2. 点击 "开发版本" 标签
3. 找到刚上传的版本

### 步骤 3：提交审核

1. 点击版本右侧的 **"提交审核"** 按钮
2. 系统会要求填写审核信息

### 步骤 4：填写审核信息

#### 基本信息

| 字段 | 填写内容 |
|------|--------|
| 小程序名称 | 五子棋 |
| 小程序简介 | 一个功能完整的微信小程序五子棋游戏，支持与 AI 对战 |
| 小程序类目 | 游戏 → 休闲游戏 |
| 小程序标签 | 五子棋、游戏、AI |

#### 功能介绍

```
本小程序是一款五子棋游戏，主要功能包括：

1. 与 AI 对手进行五子棋对战
   - AI 具有智能决策能力
   - 能够进攻和防守
   - 难度适中，游戏体验良好

2. 完整的游戏规则实现
   - 支持检测胜负
   - 支持平局判定
   - 规则清晰易懂

3. 游戏统计功能
   - 记录胜/负/平局数据
   - 数据保存在本地
   - 用户可随时查看

4. 美观的用户界面
   - 木纹棋盘设计
   - 逼真的棋子样式
   - 响应式布局适配各种屏幕

5. 完善的用户体验
   - 游戏规则说明
   - 使用提示
   - 分享功能

游戏规则：
- 黑棋先手（玩家为黑棋）
- 轮流在棋盘上放置棋子
- 先连成 5 个同色棋子（横、竖、斜）即为获胜
- 棋盘满了还没有人赢则为平局
```

#### 隐私政策

```
本小程序致力于保护用户隐私：

1. 不收集任何个人信息
   - 不获取用户微信信息
   - 不获取用户位置信息
   - 不获取用户设备信息

2. 游戏数据仅保存在本地
   - 不上传到服务器
   - 用户完全控制自己的数据
   - 用户可随时删除数据

3. 不使用第三方服务
   - 不集成广告
   - 不使用数据分析
   - 不接入社交媒体

详见：https://github.com/leeking001/wechat-gomoku/blob/master/PRIVACY_POLICY.md
```

#### 操作步骤

```
1. 打开小程序
2. 等待游戏加载（通常 2-3 秒）
3. 点击棋盘空位放置黑棋
4. 等待 AI 放置白棋
5. 轮流下棋直到游戏结束
6. 点击"重新开始"开始新局
7. 点击"游戏规则"查看详细说明
8. 点击"分享游戏"分享给朋友
```

## ⏳ 第四阶段：等待审核（24 小时）

### 审核期间

- 微信通常在 24 小时内完成审核
- 审核结果会发送到你的邮箱
- 可在后台"审核版本"中查看审核状态

### 可能的审核结果

#### ✅ 审核通过

1. 收到"审核通过"邮件
2. 后台版本状态变为"审核通过"
3. 可以进行发布

#### ❌ 审核不通过

1. 收到"审核不通过"邮件，说明原因
2. 修改代码或信息
3. 重新上传并提交审核

常见不通过原因：
- 功能不完整
- 隐私政策不清
- 体验不好
- 违规内容
